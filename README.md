# BodProject

스프링부트 기반의 개인, 그룹 SNS

##기능

---

1. 회원
    * 회원가입
    * 아이디, 비밀번호 찾기
    * 카카오 계정으로 로그인
    * 로그인
    * 로그아웃
    * 개인정보 수정
    * 친구 요청, 수락, 친구 목록에서 삭제
    * 회원 검색

2. 게시글
    * 게시글 작성(다수의 이미지, 동영상 업로드 가능, 슬라이드, 해시태그 추가)
    * 게시글 수정(기존 이미지, 동영상 파일 삭제 후 새파일 업로드, 해시태그 삭제)
    * 그룹글 작성, 수정
    * 타임라인(친구와 내가 속한 그룹의 게시물)
      - 댓글, 대댓글, 좋아요, 태그(Ajax)
      - 태그 클릭시 해시태그된 게시글
 
3. 그룹
    * 그룹 생성, 수정
    * 그룹 검색, 가입, 탈퇴
    * 그룹 페이지
      - 그룹 멤버 목록(친구 추가, 채팅)
      - 그룹 게시글 작성, 수정

4. 관리자
     * 전체 회원 목록
       - 회원 탈퇴(회원의 정보, 게시글, 댓글, 대댓글, 좋아요, 그룹, 친구목록)
     * 전체 글 목록
       - 글삭제(해당 글의 이미지, 동영상, 댓글, 대댓글, 좋아요)
     * 그룹 목록
       - 그룹삭제(그룹, 그룹멤버, 그룹게시글)
